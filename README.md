# 🎓 Attendance Intelligence Platform

> **AI-powered attendance intelligence for schools and colleges.**
>
> Transform attendance data into actionable insights that help teachers identify at-risk students, automate attendance management, and improve student engagement through intelligent analytics.

---

# 🌟 Overview

Attendance Intelligence Platform is an AI-powered web application that modernizes attendance management for educational institutions. Instead of simply recording attendance, the platform analyzes attendance patterns, identifies students who may be at academic risk, provides actionable insights to teachers, and enables proactive communication with parents.

Built for the **NxtWave Idea2Impact Hackathon 2026**.

---

# ❗ Problem Statement

Traditional attendance systems focus only on recording attendance. Most schools and colleges still rely on paper registers or basic digital systems that consume valuable classroom time and provide little actionable insight.

As a result:

- Teachers spend valuable teaching time managing attendance.
- Students with declining attendance are identified too late.
- Parents receive delayed attendance updates.
- Administrators lack institution-wide attendance analytics.
- Institutions cannot make proactive, data-driven decisions.

Attendance is one of the strongest indicators of student engagement and academic success, yet existing systems fail to leverage attendance data intelligently.

---

# 💡 Our Solution

Attendance Intelligence Platform combines digital attendance management with AI-powered analytics to transform attendance records into meaningful educational insights.

The platform enables institutions to:

- Digitally manage attendance
- Detect at-risk students early
- Generate attendance analytics
- Automate attendance reporting
- Improve teacher productivity
- Strengthen parent-school communication

# ⚙️ Setup Instructions

## 1. Clone Repository

```bash
git clone https://github.com/Vijayb24bs1408/attendance-tracker.git
```

---

## 2. Move into Project

```bash
cd attendance-tracker
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Configure Firebase

Create a `.env` file in the project root.

```env
VITE_FIREBASE_API_KEY=YOUR_API_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_SENDER_ID
VITE_FIREBASE_APP_ID=YOUR_APP_ID
```

---

## 5. Run Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
